# Voice Helper (Web)

Веб-клиент для управления задачами. Вход через Telegram Login Widget; задачи привязаны к текущему пользователю.

## Запуск

```bash
cp .env.example .env   # при наличии; задать переменные
npm install
npm run dev
```

## Переменные окружения

- `VITE_API_URL` — URL бэкенда (по умолчанию `http://localhost:3000`).
- `VITE_TELEGRAM_BOT_USERNAME` — имя бота для виджета входа (например, `MyTaskBot`). Без него кнопка «Войти через Telegram» не отображается.

## Авторизация

- Неавторизованных пользователей перенаправляет на страницу входа (`/signin`).
- Вход выполняется через [Telegram Login Widget](https://core.telegram.org/widgets/login): после успешной авторизации данные отправляются на `POST /auth/telegram`, бэкенд возвращает JWT и данные пользователя; токен сохраняется в `localStorage`.
- Все запросы к API задач отправляются с заголовком `Authorization: Bearer <token>`; отображаются и редактируются только задачи текущего пользователя.
- Выход: кнопка «Выйти» очищает токен и перенаправляет на `/signin`.
